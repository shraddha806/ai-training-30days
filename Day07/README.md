# Weather Mini Project

## Objective
Fetch live weather data for 5 Indian cities using WeatherAPI, clean data with Pandas, save to CSV, and analyze results.

## Cities Used
- Bengaluru
- Mumbai
- Delhi
- Hyderabad
- Chennai

## Technologies
- Python
- Requests
- Pandas
- CSV
- GitHub

## Output File
weather_data.csv

## Key Analysis
1. Hottest city identified
2. Most humid city identified
3. Windiest city identified
4. Average temperature calculated
5. Cities above 30°C filtered
